hexo博客的部署．更新和上线
（前提是在deepin的Linux里进行操作，hexo ,apt)

１．hexo博客的部署
首先打开deepin版本，右击打开终端，进行如下操作
npm install hexo-cli -g
apt install npm

sudo apt install npm
输入密码
sudo apt install node nodejs-bin
sudo npm install hexo-cli -g
sudo apt install git
hexo init blog
cd blog
npm install
hexo server

以上完成后当出现localhost:4000说明以上hexo博客的部署正确．

这是关键步骤．

2.hexo博客的更新
写作：

sudo apt install gedit
hexo new “title
gedit ~/…md

(写好不等于发表)hexo g
hexo d
hexo s

3.hexo博客的上线
首先百度找到Github网站，第一步注册账号。
记住网址　　https://github.com/yangxiangbo/yangxiangbo.github.io.git
进入终端，输入 cd blog
sudo npm install hexo-deployer-git –save
git config –global user.email “ “
git config –global user.name “ “
gedit _config.yml
改type: git
repository: https://github.com/yangxiangbo/yangxiangbo.github.io.git
branch: master

hexo g
hexo d

之后更新博客笔记直接cd blog
hexo new “TITLE”
gedit ~/…md
hexo g
hexo d
